# Mobile Development Environment Setup

## Objective
The goal of this setup is to prepare a suitable mobile development environment using Expo for React Native development. Expo Go allows testing applications directly on a physical device without relying on emulators.

## Tools Used
- Node.js (LTS version)
- Visual Studio Code
- Windows OS
- Expo Go (Android)
- Physical Android device
